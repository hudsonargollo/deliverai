## 🔄 Customer Journey

### Before (Old Flow)
1. Add items to cart
2. Click checkout
3. Enter name (required)
4. Enter WhatsApp (required)
5. Enter address (required)
6. Select payment
7. Confirm order

**Issues**: Name asked immediately, no cart review, no address editing

### After (New Flow)
1. Add items to cart
2. Click "View Cart"
3. Review cart items (edit quantities, remove items)
4. Enter name and WhatsApp
5. Enter delivery address (with Google Maps search)
6. Select payment method
7. Confirm order

**Improvements**: Cart review first, easy editing, better organization, clearer steps
